## Project Overview  
This is a desktop application built in Java (with Maven) that allows users to track their expenses. The code is in the repository: [DesktopExpenseTracker](https://github.com/Vishnujha44/DesktopExpenseTracker).

## Features  
- Add new expense entries (date, category, amount, description)  
- Edit / delete existing entries  
- View expense list and filter by category / date  
- Export or backup data (if implemented)  
- User-friendly GUI for seamless desktop usage  

## Technologies Used  
- Java  
- Maven (pom.xml present)  
- JavaFX or Swing (depending on GUI implementation)  
- SQL/embedded storage (if used)  
- Git for version control
